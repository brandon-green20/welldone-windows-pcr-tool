## Overview

WellDone is a Windows-based companion for PCR setup work that operates entirely offline and is centered on practical output for Roche LightCycler PRO workflows. It brings sample organization, plate layout planning, and related setup information into a single workflow so repetitive preparation steps take less manual effort.

The application is aimed at laboratory users who need dependable PCR preparation materials without depending on a network connection. With German and English language support plus a manual in both languages, it is suited to environments where clear instructions and repeatable setup procedures are important.

## What it does

- Automatic PCR plate allocation for setup planning
- Sample recording for organized workflow tracking
- Pipetting scheme PDF generation for setup documentation
- Plate setup CSV generation for data exchange and records
- Order creation support for MagNA Pure 96 workflows
- Offline operation without an internet connection
- German and English interface for broader usability
- German and English manual for guided use
